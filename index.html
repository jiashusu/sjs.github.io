开始搭建

安装 node

因为 hexo( 我们的牛逼老大哥 ) 是基于 node 框架的，所以呢，我们首先需要下载安装node，下载地址：nodejs
安装完之后，我们打开命令窗口，输入 node -v，如果返回下图所示，那么就说明你安装 node 成功了。

初始化blog
  1. 安装成功后，我们在命令行窗口运行如下命令来安装hexo：npm install hexo-cli -g
  2. 初始化博客目录：hexo init user.github.io (这里的user换成你自己的名字)
  3. 初始化完成后，我们就进入我们的目录：cd user.github.io
  4. 安装npm install
  5. clean一下：hexo clean
  6. 然后生成静态页面： hexo g

    g 就是generate, 生成的意思

  7. 接着把你的网站运行起来hexo s

    s 就是server ,在服务器运行的意思

打开你的浏览器，输入 localhost:4000 。 自此，你的个人网站就这么速度的搭建起来了！
